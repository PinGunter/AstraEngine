virtual void createDescriptorSetLayout ()
 Crea el layout de los descriptor sets básicos para rasterización. El descriptor set que crea tiene la siguiente estructura:
Binding 0: Parámetros de Cámara
Binding 1: Datos de las luces
Binding 2: Datos de los objetos, direcciones de memoria para acceder a los buffers
Binding 3: Texturas.

Atención
