\doxysubsection{Descripción detallada}
Contiene los parámetros necesarios para configurar el dispositivo (GPU). Por defecto utiliza ray-\/tracing ~\newline
Las extensiones se configuran por defecto con las necesarias para mostrar resultados por la pantalla y par el ray-\/tracing. 

\begin{DoxyWarning}{Atención}
Si no se rellenan los datos de capas de instancia y extensiones de instancia o dispositivo se asume que el usuario entiende lo que está haciendo. Esto quiere decir que el propio usuario {\itshape experto} deberá agregar las extensiones de GLFW y el resto de extensiones básicas como las de la swapchain. 
\end{DoxyWarning}
