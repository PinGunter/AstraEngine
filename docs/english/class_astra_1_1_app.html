◆ resetScene()

virtual void Astra::App::resetScene ( bool recreatePipelines = false)
protectedvirtual

Resets the scene. Should be called whenever the current scene changes or if models are added in runtime.

Warning
Calling the method while rendering will probably crash the scene as Descriptor Sets will be outdated as the shaders are running! Make sure to reset it before or after a render action.

Reimplemented in Astra::AppRT.

◆ run()

virtual void Astra::App::run ( )
pure virtual

Pure virtual method. The derived class should implement the drawing loop, the event polling (using InputManager), updating the scene and drawing it. After the loop, destroy() should be called
Example code:

void run(){
while (!Astra::Input.windowShouldClose()){
// dont render if is minimized
if (isMinimized()) continue;
auto cmdList = _renderer->beginFrame();
updateScene(cmdList);
_renderer->render(cmdList, _scenes[_currentScene], _pipelines[_selectedPipeline], { _descSet }, _gui);
_renderer->endFrame();
}
}
destroy();
